# 📊 Análise de Vendas com SQL

## 🎯 Objetivo
Analisar o desempenho de vendas por vendedor e identificar oportunidades de crescimento.

## 🗂️ Base de dados
Dados simulados de vendas contendo:
- Código do vendedor
- Nome
- Valor de venda
- Quantidade
- Data

## 🛠️ Tecnologias utilizadas
- SQL
- Excel / Power BI

## 📈 Análises realizadas
- Total de vendas por vendedor
- Ranking de vendedores
- Vendas por mês
- Ticket médio

## 💡 Insights
- Identificação dos melhores vendedores
- Meses com maior faturamento
- Possíveis oportunidades de melhoria
